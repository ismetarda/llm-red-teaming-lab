# LLM Red Teaming & Vulnerability Lab 🛡️

This project is a cybersecurity laboratory study designed to test automated **Red Teaming** and **Jailbreak** attack scenarios against Local LLM models (Mistral, TinyLlama).

## 🚀 Features

* **Giskard Framework:** Vulnerability scanning and comprehensive model analysis.
* **LangChain Integration:** Seamless communication with Ollama models hosted on a remote server (Host PC).
* **Automated Prompt Injection:** Testing the resilience of security filters by deploying restricted prompts (Jailbreaks) against the model.

## ⚠️ Disclaimer

This project was developed strictly for educational and ethical security testing purposes.
